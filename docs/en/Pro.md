<h1 align="center">Pro</h1>

A macro is a small program written in Klipper/Gcode language.

It can be called from:

- A GCODE file
- The Fluidd/Mainsail console (press the English letter `C` in Fluidd)

!!! note
    *The value in parentheses is the default value*

---

### AIR_CIRCULATION_INTERNAL

Enable internal air circulation.

---
### AIR_CIRCULATION_EXTERNAL

Enable external air circulation.

---
### AIR_CIRCULATION_STOP

Disable air circulation.
