<h1 align="center">ZMOD Macro List</h1>

A macro is a small program written in Klipper/Gcode.

It can be called from:

- A GCODE file
- The Fluidd/Mainsail console (press the `C` key in Fluidd).

!!! note
    *Default values are shown in parentheses.*
